#  GitHub Information Extractor

## Project Description
GitHub Information Extractor is a useful project designed to help **recruiters and students** easily fetch and view GitHub user profile information.  
Using a GitHub username, users can quickly extract details such as **name, bio, and other public profile information** without manually visiting the GitHub profile.

## Features
- Fetches GitHub user information using a username  
- Uses the **official GitHub API**  
- Extracts user details like **name, bio, and public profile data**  
- Helpful for both recruiters and students  
- Simple and easy-to-use interface  

## Technologies Used
- GitHub API  
- HTML  
- CSS  
- JavaScript  

## How to Use
1. Enter a GitHub username  
2. Click on the fetch button  
3. The user’s profile information will be displayed on the screen  

## Use Cases
- Helps recruiters quickly review candidate GitHub profiles  
- Allows students to analyze their own and others’ GitHub profiles
